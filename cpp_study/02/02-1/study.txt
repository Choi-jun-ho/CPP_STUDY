[Q1] 키워드 const의 의미

const int num = 10;
ㄴ 변수 num을 상수화 > 값의 변경 불가능
const int * ptr1 = &val1;
ㄴ 포인트를 이용하여 val1의 값을 변경할 수는 없지만 ptr1의 주소 값을 변경할수 있음 하지만 그 주소가 쓰래기 값일 수도 있으니 주의
int * const ptr2 = &val2;
ㄴ 포인트를 이용하여 val2의 값을 변경할 수는 있지만 주소값은 변경불가
constt int * const ptr3 = &val3;
ㄴ 포인터 ptr3가 상수화 되었으며, ptr3를 이용해서 val3의 값을 변경할 수 없음

[Q2] 실행중인 프로그램의 메모리 공간
실행중인 프로그램은 운영체제로부터 메모리 공간을 할당 받는데, 이는 크게 데이터, 스택, 힙 영역으로 나뉜다
각각의 영역에는 어떠한 형태의 변수가 할당되는지 설명해보자. 특히 C언어의 malloc과 free함수와 관련해서도 설명해보자

데이터 : 전역변수가 저장되는 영역, const, static을 이용한 변수가 여기에 저장
스택 : 지역변수 및 매개변수가 저장되는 영역
힙 : malloc함수호출에 의해 프로그램이 실행되는 과정에서 동적으로 할당이 이루어지는 영역
malloc & free : malloc 함수호출에 의해 할당된 메모리 공간은 free 함수호출을 통해서 소멸하지 않으면 해재되지 않는다.

A 메모리 주소를 사용하는가 안하는가에 있다
swap 함수를 예를 들자면 참조에이한호출은 포인터로 이용하여 메개변수에 직접적인 메모리 주소가 들어가 있어 리턴값없이 함수안에서 바로 적용할수 있지만
값에 의한 호출은 리턴값을 주고 각각 다시 초기화 시켜야지 변경이 된다
